This is a form based interface to GlimpseHTTP WWW search engine. Glimpse is an indexing/search program that uses very little space for the index and allows flexible queries. This glimpse interface allows both browsing and searching. A search covers all files in this directory and all directories below it. As you browse deeper, you automatically limit the search to where you are.

For information about Glimpse and where to obtain a copy (for your personal file system or for your server) follow this link or mail glimpse@cs.arizona.edu
